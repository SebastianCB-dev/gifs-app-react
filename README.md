# Gifs App React
 
## Description
This is a simple app that shows gifs from the Giphy API. It was made with React and some hooks.

## Installation
To install this app you need to clone the repository and run the following command in the root folder:

```bash
pnpm install
```

or

```bash
npm install
```

then 
  
```bash
pnpm vite
```
or
```bash
npm run vite
```
